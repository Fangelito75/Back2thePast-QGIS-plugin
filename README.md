# Back2thePast

Back2thePast is a QGIS plugin that enables users to explore and load historical satellite imagery from Esri World Imagery Wayback, organised by date.

It allows easy temporal analysis of landscape changes by providing quick access to different imagery releases and facilitating comparison across time.

## Features

- Browse Esri Wayback imagery by date
- Load the latest available imagery automatically
- Filter imagery by year or keyword
- Group loaded layers in QGIS
- Clear plugin-loaded layers easily

## Installation

1. Open QGIS
2. Go to Plugins → Manage and Install Plugins
3. Install from ZIP
4. Select the plugin package
